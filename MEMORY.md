- **核心理念**：周期到了自然就涨，找出处于低位的板块，预判资金轮动方向
- **文档位置**：`/root/.openclaw/workspace/周期战法策略.md`
- **创建时间**：2026-04-02

**主要分析维度**：
1. 行业板块 + 概念板块（共约1000个）
2. 涨跌幅分析（高位/低位）
3. 换手率（资金活跃度）
4. 资金流向（主力净流入/流出）

**低位板块特征**：

- **范围**：全球金融市场（中、美、欧、亚太）
