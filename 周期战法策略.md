# 周期战法 - A股板块轮动策略

## 策略概述

**核心理念**：周期到了自然就涨。找出处于低位的板块，预判资金轮动方向，在启动前布局。

**数据来源**：东方财富、同花顺、AkShare

---

## 一、数据获取

### 1. 行业板块数据
```python
import akshare as ak
industry_df = ak.stock_board_industry_name_em()  # 获取所有行业板块
```

### 2. 概念板块数据
```python
concept_df = ak.stock_board_concept_name_em()  # 获取所有概念板块
```

### 3. 板块资金流向
```python
# 行业资金流
fund_flow_ind = ak.stock_sector_fund_flow_rank(indicator="今日", sector_type="行业资金流")
# 概念资金流
fund_flow_concept = ak.stock_sector_fund_flow_rank(indicator="今日", sector_type="概念资金流")
```

## 二、分析维度

### 1. 涨跌幅分析
- **高位板块**：涨幅 > 2%，可能过热
- **低位板块**：跌幅 > 2%，可能超跌
- **活跃板块**：换手率 > 2%

### 2. 资金流向
- 主力净流入/净流出
- 主力净流入占比

### 3. 周期位置判断
- 今日表现（涨幅/跌幅）
- 换手率（资金活跃度）
- 近几日表现（趋势）

---

## 三、板块分类标准

### 高位板块（谨慎追高）
| 类型 | 涨幅 | 换手率 | 风险等级 |
|------|------|--------|----------|
| 短线热点 | > 5% | 高 | ⚠️ 极高 |
| 资金追捧 | 3-5% | > 3% | ⚠️ 高 |
| 温和上涨 | 1-3% | 1-2% | 中 |

### 低位板块（潜在机会）
| 类型 | 跌幅 | 换手率 | 机会等级 |
|------|------|--------|----------|
| 超跌 | > 5% | > 2% | 💎 高 |
| 调整 | 2-5% | > 1.5% | 💎 中 |
| 震荡 | < 2% | < 1% | 观察 |

---

## 四、资金轮动预判逻辑

### 1. 领涨板块特征
- 涨幅最大 + 换手率高 = 资金主攻
- 但要注意：涨太多可能轮到别的板块

### 2. 资金轮动规律
- 资金从高位板块撤出 → 流入低位板块
- 热点板块调整 → 资金寻找新方向
- 政策利好 → 资金提前布局

### 3. 预判信号
- **板块A持续上涨后高位盘整** → 资金可能轮动到相关板块
- **某板块持续下跌但换手率不低** → 可能在吸筹
- **消息面变化** → 资金提前反应
