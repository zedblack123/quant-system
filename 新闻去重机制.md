# 新闻去重与价值判断机制

## 核心逻辑

### 1. 去重机制
- 使用新闻Hash或标题作为唯一标识
- 已推送的新闻记录到 `news_pushed.json`
- 同一新闻7天内不重复推送

### 2. 价值判断标准

#### ✅ 值得推送的新闻
| 类型 | 标准 |
|------|------|
| **涨停/跌停** | 首次出现的新标的 |
| **龙头股异动** | 涨幅>9%或跌幅>7% |
| **政策利好/利空** | 官方媒体确认的消息 |
| **行业重大利好** | 首次出现的行业政策 |
| **外资大额流入/流出** | 单日净流入/流出超5亿 |
| **大盘剧烈波动** | 指数涨跌幅>1.5% |
| **炸板/天地板** | 首次炸板 |

#### ❌ 不推送的内容
| 类型 | 原因 |
|------|------|
| 传闻/传言 | 未证实，不可靠 |
| 重复新闻 | 7天内已推送过 |
| 旧闻 | 超过24小时 |
| 无实质内容 | 只有标题没有详情 |
| 市场正常波动 | 无需特别关注 |

### 3. 推送间隔
- 正常情况：30分钟检查一次，有价值新闻才推送
- 异常市场：大盘剧烈波动时，即时推送预警
- 早间/午间：可适当增加推送密度

### 4. 已推送新闻记录格式
```json
{
  "pushed_news": [
    {
      "title": "新闻标题",
      "hash": "标题MD5哈希",
      "pushed_at": "2026-04-02T10:00:00+08:00",
      "type": "limit_up|policy|sector|moneyflow|...",
      "source": "同花顺|东方财富|..."
    }
  ]
}
```
